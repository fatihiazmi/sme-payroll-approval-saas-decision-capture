Start with `README.md`, then review these files in order:

1. `docs/decision-capture/payroll-cosec-workflow.md`
2. `docs/decision-capture/finance-audit-evidence.md`
3. `docs/decision-capture/saas-product-architecture.md`

- Accepted
- Changed
- Rejected
- Needs Client Confirmation
